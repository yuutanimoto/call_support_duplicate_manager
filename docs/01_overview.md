# プロジェクト概要

## システム概要
PostgreSQLデータベースから取得した受信データの重複を検出・管理するWebアプリケーションです。管理者が重複データを確認・削除できるインターフェースを提供します。

## 主要機能

### 1. 重複データ検出
- **完全一致重複（exact）**: 受付内容（rdata）と対応状況（execstate）の両方が一致するデータを検出
- **受付内容重複（content）**: 受付内容（rdata）のみが一致するデータを検出  
- **対応状況重複（status）**: 対応状況（execstate）のみが一致するデータを検出

### 2. データ管理機能
- **データ表示**: ページネーション対応のテーブル表示
- **高度なフィルタリング**:
  - 分離型キーワード検索（受付内容・対応状況を個別検索）
  - 統合キーワード検索（後方互換性）
  - 進捗、システム種別、製品フィルター
  - 日付範囲フィルター（受付日時/更新日時/削除日時）
  - 削除済みデータの表示制御
- **ソート機能**: 各列での昇順・降順ソート
- **論理削除**: データを物理削除せず、削除フラグ（receptmoddt）を設定
- **データ復元**: 削除済みデータを元の状態に復元
- **統計情報**: 全体・有効・削除済みデータ数をリアルタイム表示

### 3. Webインターフェース
- **レスポンシブ対応**: PC専用設計（1200px以上推奨）
- **リアルタイム操作**: Ajax通信による非同期処理
- **バッチ操作**: 複数データの一括削除・復元
- **スマートフィルタリング**: 検索条件に応じた動的データ更新

## 技術仕様

### バックエンド
- **フレームワーク**: FastAPI 0.104.1
- **言語**: Python 3.8+
- **データベース**: PostgreSQL
- **ORM**: psycopg2-binary 2.9.7（生SQL使用）
- **バリデーション**: Pydantic 2.4.2
- **テンプレート**: Jinja2 3.1.2

### フロントエンド
- **HTML5** / **CSS3** / **JavaScript (ES6+)**
- **バニラJS**: フレームワーク非依存の実装
- **モジュール構成**: 機能別ファイル分割

### インフラ
- **Webサーバー**: Uvicorn 0.24.0
- **環境管理**: python-dotenv 1.0.0
- **CORS対応**: 全オリジン許可設定

## システム要件
- Python 3.8以上
- PostgreSQL データベース
- 1200px以上の画面解像度（推奨）

## 対象データ範囲
- 2015年1月1日以降のデータ
- extentid が NULL でない有効なレコード
- アクティブデータ（receptmoddt が NULL）および削除済みデータ
  - デフォルト: アクティブデータのみ表示
  - オプション: include_deletedパラメータで削除済みデータを含む

## セキュリティ考慮事項
- データベース認証情報の環境変数管理
- 論理削除による完全なデータ復旧可能性
- SQLインジェクション対策（パラメータ化クエリ）
